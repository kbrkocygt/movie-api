# Movie API (.NET)

This project is a backend API built with ASP.NET Core using Clean Architecture and CQRS pattern.

## Architecture
- Clean Architecture (Core, Infrastructure, Presentation)
- CQRS Design Pattern

## Tech Stack
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server

## Features
- Movie CRUD operations
- CQRS-based command/query separation
- Scalable and maintainable structure

## Purpose
This project was built to improve backend development skills and understand modern architecture patterns like Clean Architecture and CQRS.

## What I Learned
- Structuring scalable backend systems
- Implementing CQRS pattern
- Separation of concerns in real-world projects
